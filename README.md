# Assignment_botmakers


a full stack web app with jwt and role based access

# tech stack 
## backend: 
- Java 17, Spring Boot, Spring Security (JWT), Spring Data JPA, PostgreSQL
##frontend 
- React, TypeScript, Vite, TailwindCSS, React Query, React Hook Form


# setup 
 - setup using docker compose 

## prerequisites
 - Docker and Docker Compose

## running the app 

1. clone the repo and navigate to root directory.
2. run 

```bash
docker compose up --build
